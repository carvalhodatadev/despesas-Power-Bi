# Projeto: Dashboard de Despesas Pessoais (2022-2024)

## 📌 Visão Geral
Este projeto consiste em um **dashboard interativo** desenvolvido no **Power BI** para análise aprofundada de despesas pessoais ao longo de três anos.
O objetivo é fornecer insights sobre hábitos de consumo, categorização de gastos e evolução financeira, permitindo um controle orçamentário mais eficiente.

## 🎨 Design e Experiência do Usuário (UX)
Antes da construção no Power BI, o projeto passou por uma etapa de **prototipagem no Figma**, garantindo:
- Layout limpo e intuitivo
- Hierarquia visual bem definida
- Cores harmoniosas e acessíveis
- Fluxo de navegação claro entre as páginas

O resultado é um dashboard com **tooltips personalizados** que enriquecem a experiência do usuário, exibindo informações contextuais ao passar o mouse sobre os gráficos.

## 🛠️ Ferramentas Utilizadas
- **Power BI Desktop** – construção dos visuais e medidas DAX
- **Figma** – prototipagem e design de interface
- **Excel** – fonte de dados original
- **Git/GitHub** – versionamento e portfólio

## 📊 Funcionalidades do Dashboard
- **Visão Executiva (Home)** com KPIs principais
- **Evolução mensal dos gastos** (2022-2024)
- **Ranking das 5 maiores categorias**
- **Classificação Essencial vs. Não Essencial**
- **Análise por Categoria e Forma de Pagamento**
- **Tooltips informativos** em todos os gráficos principais
- **Tabela de detalhamento** com filtros interativos

## 🧠 Principais Insights (exemplos)
- Os gastos com **Transporte** representam a maior fatia do orçamento
- A proporção entre gastos essenciais e não essenciais é de **39,6% / 60,4%**
- O **Pix** tem mostrado tendência de queda nos últimos meses
- Os meses de **janeiro e julho** apresentam picos de despesas (férias?)

## 🚀 Como Visualizar o Projeto
1. Clone este repositório
2. Abra o arquivo `03_dashboard/Dashboard_Despesas.pbix` no Power BI Desktop
3. Explore as páginas e passe o mouse sobre os gráficos para ver os tooltips
4. Utilize os filtros para interagir com os dados
